# Grid

`Grid` is a primitive layout component that can arrange content in a grid configuration.

### Usage

```jsx live
<Grid columns={3}>
  <View css={[ui.background.blue]}>
    <Text>Ana</Text>
  </View>
  <View css={[ui.background.blue]}>
    <Text>Elsa</Text>
  </View>
  <View css={[ui.background.blue]}>
    <Text>Olaf</Text>
  </View>
</Grid>
```
