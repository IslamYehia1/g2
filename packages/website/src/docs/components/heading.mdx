# Heading

`Heading` renders headings and titles using the library's typography system.

## Usage

```jsx live
import { Heading } from "@wp-g2/components"

function Example() {
  return <Heading>Into The Unknown</Heading>
}
```

## Props

`Heading` uses `Text` underneath, so we have access to all of `Text`'s props. For more information, check out [`Text`](/components/text/).

| Name          | Type               | Default | Description                                                          |
| ------------- | ------------------ | ------- | -------------------------------------------------------------------- |
| align         | `string`           |         | Adjusts the text alignment.                                          |
| as            | `string`           | `span`  | Renders as another HTML element (or React component).                |
| color         | `string`           |         | Adjust the text color.                                               |
| display       | `string`           |         | Adjusts the text CSS `display`.                                      |
| ellipsis      | `string`           | `...`   | The ellipsis string when `truncate` is set.                          |
| ellipsizeMode | `string`           | `auto`  | Determines where to truncate. `truncate` must first be set.          |
| limit         | `number`           | `0`     | Determines the max characters when `truncate` is set.                |
| isBlock       | `boolean`          |         | Sets `Text` to have `display: block`.                                |
| lineHeight    | `number`           |         | Adjusts all text line-height based on the typography system.         |
| size          | `number`, `string` | `3`     | Adjusts all text size based on the typography system.                |
| truncate      | `boolean`          |         | Enables text truncation.                                             |
| numberOfLines | `boolean`          |         | Determines how many lines to truncate. `truncate` must first be set. |
| upperCase     | `boolean`          |         | Uppercases the text content.                                         |
| variant       | `string`           |         | Adjusts style variation of the text.                                 |
| weight        | `number`           |         | Adjusts font-weight of the text.                                     |

### size

`Heading` will typically render the sizes `1`, `2`, `3`, `4`, `5`, or `6`, which map to `h1`-`h6`. However, it can render any size, including non `px` values.

```jsx live
import { Heading, View } from "@wp-g2/components"

function Example() {
  return (
    <View>
      <Heading size={1}>Into The Unknown</Heading>
      <Heading size={2}>Into The Unknown</Heading>
      <Heading size={3}>Into The Unknown</Heading>
      <Heading size={4}>Into The Unknown</Heading>
      <Heading size={5}>Into The Unknown</Heading>
      <Heading size={6}>Into The Unknown</Heading>
    </View>
  )
}
```
